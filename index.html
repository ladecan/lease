4

금액 불일치 경고가 뜰 수 있어요

관리자가 미리 예상 커미션을 입력해둔 경우, 내가 계산한 금액과 다르면 경고창이 나타납니다. 저장 버튼을 누르면 관리자가 미리 입력한 예상 금액과 자동으로 비교됩니다.

관리자 예상값이 없으면 팝업 없이 바로 저장돼요
꼭 알아두세요
⚠️

DCD 박스 수 · 워런티 연수 직접 확인

기본 1박스는 차감 없고 추가 박스부터 차감돼요. 워런티는 자동으로 세팅되지 않아요. 실제 계약한 총 워런티 연수를 직접 선택해야 해요!

💹

수수료율 한 번 더 확인

딜러사에 따라 자동 세팅되지만 맞는지 꼭 확인해주세요.
TMI · Innomed · SevenU → 10% / 그 외 → 7%

🏢

휴먼홀딩스 기준 자동 적용

Human Holdings 딜러사일 경우 DCD 2박스까지 차감 없음 등 별도 기준이 자동 적용돼요.

💾

저장 버튼 꼭 누르기

계산만 하고 저장 안 하면 관리자에게 안 보여요. 반드시 "저장하기" 버튼까지 눌러주세요!

자주 묻는 질문
내 이름이 없어요
이번 달 데이터가 아직 업로드되지 않았거나 이름이 다르게 등록됐을 수 있어요. 관리자에게 문의해주세요.
계약금액이 실제와 달라요
계약금액은 엑셀 데이터 기준으로 자동 입력되며 수정이 불가해요. 금액이 다르다면 관리자에게 문의해주세요.
저장했는데 다시 들어가면 없어요
정상이에요! 내 이름 → 거래처 순서로 다시 들어가면 ✓ 저장됨 표시와 함께 이전에 입력한 값이 그대로 나와요.
잘못 저장했어요
다시 같은 거래처로 들어가서 수정하고 "결과 다시 저장하기" 버튼을 누르면 덮어씌워져요.
저장을 아예 취소하고 싶어요
저장된 거래처로 들어가면 저장 버튼 아래에 "저장 취소" 버튼이 나타나요. 누르면 저장된 결과가 삭제되고 미완료 상태로 돌아가요.
모바일에서도 되나요?
네! 모바일에서도 동일하게 사용할 수 있어요.
문의사항은 관리자에게 연락해주세요.
